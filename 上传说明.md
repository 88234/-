# GitHub Pages 上传说明

把本文件夹中的所有内容上传到 GitHub 仓库根目录。

仓库根目录应直接包含：

- `index.html`
- `styles.css`
- `app.js`
- `README.md`
- `.nojekyll`
- `.github/workflows/pages.yml`

上传后进入仓库：

1. 打开 `Settings`。
2. 打开 `Pages`。
3. 如果页面提示选择发布源，选择 `GitHub Actions`。
4. 等待 `Actions` 中的部署任务完成。

项目网址通常是：

`https://88234.github.io/-/`

如果你以后新建英文仓库，例如 `qinghang-ai-assistant`，网址通常是：

`https://88234.github.io/qinghang-ai-assistant/`

注意：不要把真实学生数据上传到 GitHub。网页会在浏览器本地读取 Excel，使用者应自行导入自己的脱敏数据。
