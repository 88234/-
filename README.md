# 课外智辅轻航自动化应用

这是“方向三：AI 助力课外全场景减负增效”的本地演示应用，重点模拟教师真实使用流程。

## 使用方法

直接双击打开 `index.html`。

推荐演示流程：

1. 先在“第一步：学情数据导入与干预建议”中填写班级总人数、纳入分析人数、班级画像和学生脱敏数据，或点击“载入完整演示样例”。
2. 点击“分析学情数据”，形成班级画像、重点关注学生和干预优先级。
3. 再选择五个工作舱中的任一场景，点击“生成本舱对策建议”。
4. 查看成果工作台中的“对策建议”，按真实班级情况修订为教师定稿。
5. 完成事实核查、隐私脱敏、表达适切、教师最终判断四项复核。
6. 点击“确认采用并保存记录”，教师定稿才会进入应用记录。
7. 点击“导出应用记录包”和“导出申报报告定稿”，整理后作为参赛附件材料。

如果使用 Excel 模板：

1. 在“直接导入 Excel 模板”处选择 `学情分析.xlsx`。
2. 点击“读取 xlsx 并填入下方数据”。
3. 系统会读取模板中的姓名、年级、性别、家庭情况、近30天出勤、心理咨询次数、自述心理、食堂消费和违纪情况。
4. 姓名会自动替换成 S001、S002 等脱敏编号。
5. 导入后会自动填入下方数据并生成风险分层分析。

## 这版新增能力

- 对策建议、教师定稿、定稿说明分离，避免未经审核的内容直接进入证据。
- 学情分析前置；未完成学情分析时，系统不会生成后续工作舱任务。
- 教师四项复核签核。
- 应用记录中加入材料价值说明。
- 一键导出应用记录包。
- 一键导出申报报告定稿。

## 数据安全说明

当前版本为本地离线演示版，不上传任何数据。正式参赛建议全部使用脱敏样例，不输入学生真实姓名、联系方式、家庭信息等敏感内容。
